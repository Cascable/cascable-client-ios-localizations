Bevor wir loslegen

  1. Prüfen Sie, ob Ihre Kamera von Cascable unterstützt wird.
  2. Prüfen Sie, ob die Firmware-Version Ihrer Kamera aktuell ist.
  3. Prüfen Sie, ob Sie die neueste Version von Cascable haben.

Cascable mit Ihrer WLAN-Kamera verbinden

  1. Drücken Sie die Menu-Taste auf Ihrer Kamera und wechseln Sie zu Wi-Fi im Setup-Menü.

  2. Wählen Sie die Option Wi-Fi-Funktion.

  3. Drücken Sie auf Neue Verbindung, wenn Sie diese Funktion zum ersten Mal nutzen. Für künftige Verbindungen lesen Sie bitte unten den Hinweis.

  4. Wählen Sie Aufnahme & Ansicht über Fernbedienung im nächsten Bildschirm.

  5. Öffnen Sie auf Ihrem iOS-Gerät Einstellungen > WLAN, wählen Sie Ihr WLAN-Netzwerk aus und geben Sie das Passwort ein, das auf dem Bildschirm Ihrer Kamera angezeigt wird.
  6. Starten Sie Cascable auf Ihrem iOS-Gerät. Die App wird Ihre Kamera automatisch erkennen und eine Verbindung herstellen. Je nach Kamera kann es sein, dass Sie das Pairing auf dem Bildschirm bestätigen müssen.


    7. Hinweis
    Wenn Sie das WLAN mit einem iOS-Gerät bereits früher schon verwendet haben, wählen Sie Wählen Sie einen Zielort aus dem Verlauf aus dem Menü Wi-Fi Funktion.

    Wählen Sie einfach die Option Direkt. Während die Kamera Verbindet… anzeigt, fahren Sie oben mit Schritt 5 fort. Sie sollten das Kennwort nicht erneut eingeben müssen.

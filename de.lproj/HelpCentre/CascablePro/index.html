Mit den PRO-Paketen von Cascable erschließt sich Ihnen das gesamte Potenzial Ihrer WLAN-Kamera.

Kaufen

Das PRO-Paket können Sie im Cascable Store erwerben. Wählen Sie dazu die entsprechende Zeile im Menü „Einstellungen“ oder tippen Sie auf eine beliebige Funktion, die mit dem ★PRO Symbol gekennzeichnet ist.

Im Cascable Store tippen Sie auf Kaufen und melden Sie sich mit Ihren Apple ID-Zugangsdaten oder mit Touch ID an.


Einkäufe wiederherstellen

Wenn Sie bereits eine oder mehrere Pro Funktionen von Cascable gekauft haben und sie auf einem anderen Gerät mit demselben Apple ID wiederherstellen möchten, wechseln Sie zum Cascable Store und tippen Sie unten auf der Seite auf Einkäufe wiederherstellen.

Komplettes Paket

Mit dem kompletten Paket bekommen Sie alle Pro-Funktionen, die Cascable zu bieten hat, zu einem günstigeren Preis.

Fernbedienung

Mit dem Fernbedienung Paket erhalten Sie Zugriff auf erweiterte Funktionen, mit denen Sie Ihre Kamera effektiv per Fernbedienung steuern.

  • Schnellansicht
    • Mit der Schnellansicht können Sie direkt im Anschluss eine Vorschau Ihres Fotos in hoher Auflösung darstellen, um sicher zu sein, dass das Ergebnis stimmt.
  • Fokus-Peaking
    • Es wird eine rote Hervorhebung um den Bereich des Suchfensterbilds gezeichnet, der scharfe Kanten hat, damit genau die richtigen Stellen des Fotos gestochen scharf werden.
  • Shutter Robot
    • Sie möchten einen Intervalometer für Zeitrafferaufnahmen stellen, Belichtungs-Bracketing für HDR einrichten, einen Bulb-Timer für Nachtaufnahmen konfigurieren oder einfach nur den Selbstauslöser aktivieren? Der Shutter Robot steht Ihnen für alle diese Aufgaben zur Seite.

Fotomanagement

Mit dem Fotomanagement-Paket erhalten Sie zahlreiche zusätzliche Tools wie zum Beispiel:

  • Bildvorschau auf Kamera
    • Hoch auflösende Vorschaubilder Ihrer Fotos direkt auf Ihrer Kamera darstellen.
  • Unterstützung für RAW-Dateien
    • Vorschau, Kopieren und Teilen von RAW-Dateien. Ganz egal ob Canon (CR2), Nikon (NEF), Olympus (ORF) oder Sony (ARW) RAW – Cascable kommt mit allen diesen Formaten zurecht.
  • Filtern und Suchen
    • Auf Ihr iPhone oder iPad kopierte Fotos nach Dateityp oder Datumsbereich filtern oder per Freitextsuche Aufnahmen finden, die mit einem bestimmten Kamerahersteller, -modell, Objektiv- oder Dateinamen gekennzeichnet sind.
  • Externe Displays
    • Sehen Sie Ihre Fotos auf dem großen Bildschirm an. Sie können sie über AirPlay streamen oder Ihr Display über einen HDMI- oder VGA-Adapter anschließen.

Nachtmodus

Mit dem Nachtmodus Paket können Sie die Bildschirmdarstellung auf dunklere Farben umstellen, um Ihre Augen beim Arbeiten im Dunkeln zu schonen. In diesem Modus wird die Benutzeroberfläche verdunkelt und Ihre Fotos rücken in den Vordergrund, sodass Sie sich voll auf das Fotografieren von Nachtaufnahmen konzentrieren können.
